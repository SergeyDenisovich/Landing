# [Landing page](https://sergeydenisovich.github.io/test/)

### В проекте используются следующие технологии:
- HTML
- SCSS
- javascript
- библиотека jquery
- fancybox
- slick-slider

Скачать репозиторий:

```sh
git clone git@github.com:SergeyDenisovich/site.git
```

Установить зависимости:

```sh
npm install
```

Запуск проекта:

```sh
npm run dev
```
